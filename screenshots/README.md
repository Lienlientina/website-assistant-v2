# 專案截圖

請將以下截圖放置在此資料夾：

## 建議的截圖內容：

### 1. `popup.png` - Extension Popup 介面
- Extension 打開後的主介面
- 顯示對話框、輸入框、按鈕等 UI 元素

### 2. `chat-example.png` - 對話範例
- 實際的對話截圖
- 最好展示 RAG 效果（如：「生病要附診斷書嗎？」的回答）
- 展示 AI 如何理解口語化問題並給出準確答案

### 3. `screenshot-feature.png` - 截圖分析功能
- 展示如何使用 📷 按鈕截取網頁
- 顯示圖片預覽和對話

### 4. `rag-demo.png` - RAG 檢索效果（可選）
- 展示系統如何從知識庫檢索相關資訊
- 可以是後端 console 的輸出或對話效果

## 截圖規格建議：

- **格式**: PNG（支援透明背景）
- **尺寸**: 建議寬度 800-1200px
- **品質**: 高清，避免模糊
- **內容**: 確保沒有個人隱私資訊

## 使用方式：

截圖放好後，README.md 中的專案預覽區塊會自動顯示：

```markdown
<p align="center">
  <img src="screenshots/popup.png" width="400" alt="Extension介面" />
  <img src="screenshots/chat-example.png" width="400" alt="對話範例" />
</p>
```
