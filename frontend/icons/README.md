# Chrome Extension 圖標

請將以下尺寸的圖標放置在此資料夾：

- icon16.png (16x16 像素)
- icon48.png (48x48 像素)  
- icon128.png (128x128 像素)

## 建議

你可以使用任何圖片編輯工具創建圖標，或使用線上工具：
- https://www.favicon-generator.org/
- https://www.canva.com/

## 臨時方案

如果暫時沒有圖標，Extension 仍可正常運行，只是會顯示預設圖標。

## 設計建議

- 使用簡單、易識別的設計
- 選擇與 AI 助理主題相關的圖案（如機器人、對話泡泡等）
- 確保在小尺寸下仍清晰可見
- 使用透明背景的 PNG 格式
